# SID Project 2 – Java Web Application

## Technologies
- Java
- JSP / Servlets
- Tomcat 10
- MySQL
- JDBC

## Setup Instructions

1. Clone the repository
2. Import project into Eclipse (Existing Project)
3. Configure Tomcat 10
4. Create MySQL database using `database.sql`
5. Add MySQL Connector/J to `WEB-INF/lib`
6. Update DB credentials in `DBConnection.java`
7. Run the project

## Default Roles
- client → index.jsp
- admin → admin.jsp (to be implemented)
